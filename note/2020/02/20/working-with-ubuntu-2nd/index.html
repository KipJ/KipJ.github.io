Working with Ubuntu (2)
VSCode, sublime, VMware and miniconda
2020-02-20 12:23   Linux   Ubuntu   1 min 19 s.

如何在 Ubuntu 中安装一些开发软件。

VScode

  1. VSCode 的安装非常简单
$ sudo snap install --classic code
code a622c65b from Visual Studio Code (vscode✓) installed
  1. 打开
$ code

Sublime Text 3

Reference：Linux Package Manager Repositories

  1. Install the GPG key:
wget -qO - https://download.sublimetext.com/sublimehq-pub.gpg | sudo apt-key add -
  1. Ensure apt is set up to work with https sources:
sudo apt-get install apt-transport-https
  1. Select the channel to use:

Stable

echo "deb https://download.sublimetext.com/ apt/stable/" | sudo tee /etc/apt/sources.list.d/sublime-text.list

Dev

echo "deb https://download.sublimetext.com/ apt/dev/" | sudo tee /etc/apt/sources.list.d/sublime-text.list
  1. Update apt sources and install Sublime Text
sudo apt-get update
sudo apt-get install sublime-text
  1. Open
$ subl

VMware Workstation Pro

  1. 下载安装包，VMware Workstation Pro Download

  2. 安装

$ chmod +x VMware-Workstation-Full-15.5.1-15018445.x86_64.bundle 
$ ./VMware-Workstation-Full-15.5.1-15018445.x86_64.bundle
  1. 在 Ubuntu 的 菜单中点击 VMware 的图标启动

这时可能汇报错：

Build environment error! ...

你需要安装依赖包：

$ sudo apt install gcc g++ make
  1. license for VMware Workstation Pro 15

任选一个输入

go ahead

miniconda

  1. 下载安装包，miniconda3

  2. 安装

$ sh Miniconda3-latest-Linux-x86_64.sh

其中选择如下：

Do you accept the license terms? [yes|no]
[no] >>> yes
Do you wish the installer to initialize Miniconda3
by running conda init? [yes|no]
[no] >>> yes
  1. 使能

重启终端，或者使用命令

$ source ~/.bashrc
