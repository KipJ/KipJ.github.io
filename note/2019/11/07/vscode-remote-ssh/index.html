Visual Studio Code: Remote-SSH

Kip · 2019-11-07 · Words: 49

  1. 第一步当然是先在 VSCode 中安装 Remote-SSH 扩展啦

  2. 通过 WSL 生成公私钥对

$ ssh-keygen -t rsa -b 4096
  1. 通过 ssh 登录服务器，将生成的公钥文件的副本 id_rsa.tmp.pub 复制到路径 ~/.ssh/ 下，执行命令
$ cat id_rsa.tmp.pub >> authorized_keys
$ chmod 600 authorized_keys

Note：必须执行 chmod 600 authorized_keys 命令

  1. 配置本地的 VSCode

编辑文件 C:\Users\<UserName>\.ssh\config，按如下格式配置：

Host arbitrary-name-you-like
    HostName remote-host-ip
    Port remote-port
    User user-name-to-sign-in
    IdentityFile C:\Users\<UserName>\.ssh\id_rsa
