Install Qv2ray in Ubunut

July 11, 2020   Rainer Chiang     2 min 39 s  

SSR

之前用的 VPS 到期，而重新搭建服务太麻烦，就直接购买别人已经搭建好的 SSR 服务。

但是，服务商提供的小飞机 ShadowsocksR 不好用，我将它换成了 Qv2ray。

已经使用几周，体验还不错。

Qv2ray

Qv2ray 运行在 v2ray 内核基础上，通过插件的形式提供对 SSR 的支持。

Download Softwares

需要下载三样东西：

  1. Qv2ray 客户端，我为 Ubuntu 下载的是 Qv2ray.v2.6.0.linux-x64.AppImage。AppImage 格式的文件，只需为其添加运行权限即可，免去安装各种依赖的痛苦。

  2. v2ray-core 内核，我这里下载 v2ray-linux-64 v4.26.0.

  3. QvPlugin-SSR 插件，下载 QvSSRPlugin.v2.0.0.linux-x64.so 正式版，其刚好适配 Qv2ray v2.6.0 正式版。

Configure Qv2ray

  • 可执行权限

将 Qv2ray.v2.6.0.linux-x64.AppImage 放置在 ~/.config/qv2ray/ 路径下，并赋予其可执行权限：

$ chmod +x Qv2ray.v2.6.0.linux-x64.AppImage
  • Add to Launcher

编辑应用配置文件：/usr/share/applications/qv2ray.desktop：

$ sudo vim /usr/share/applications/qv2ray.desktop

配置如下：

[Desktop Entry]
Name=Qv2ray
Name[zh_CN]=Qv2ray
Comment=VPN
Comment[zh_CN]=VPN
Exec=/home/kip/.config/qv2ray/Qv2ray.v2.6.0.linux-x64.AppImage
Icon=/home/kip/.config/qv2ray/qv2ray.png
Terminal=false
Type=Application
Categories=Application;VPN;
Encoding=UTF-8
StartupNotify=true

其中的 ‘Icon’，可以去其 GitHub 主页下载。

保存退出后，就可以在 Dash 中搜索 qv2ray 并将之添加到快速启动栏。

Configure v2ray-core

将 v2ray-linux-64.zip 加压到 ~/.config/qv2ray/vcore/ 路径下，并在 「Preferences –> Kernel Settings」下，点击 「Check V2ray Core Settings」 检查是否通过。

Configure SSR Plugin

将插件 QvSSRPlugin.v2.0.0.linux-x64.so 复制到路径 ~/.config/qv2ray/plugins/ 下。

重新启动 Qv2ray 客户端，点击 「Plugins」，可以看到 SSR 插件已经被识别到了。

Add Subscription

点击 Qv2ray 主界面的 「Groups/Subscription」，将你的订阅地址填入并点击更新。

待更新完成后，在主界面双击相应的节点，即可启动代理。
